JavaScript Individual Task

Yeeeehaaaa. It's JavaScript time. This week we will be learning about the beauty of JavaScript & JQuery. We will also deal with our first API, and learn how to use it, manipulate data, and display it.

Pretend you are a new intern working for Rocket Internet, and you are in charge of creating a clone of Instagram.

Here is an example: Instagram.ly™

The required features are:
  • When the user types a hashtag in the search bar and clicks "search", a list of instagram posts for that hashtag is shown.
  • Be hosted online.

Feel free to design it however you want!

Extras:
  • When a search is happening, a "loading" icon is shown (in my example, you can see a spinning animation). You can choose any icon you want.
  • Be responsive
  • A user can press "enter" to search instead of clicking the "search" button.
  • Clicking on a post's username brings you to their instagram profile
  • Show who posted it, and their avatar
  • Show the comments & like count for each instagram post
  • Add a page where you can search for a user and display all their posts.
